Bot for discord

This project is an improvisation, and was not originally intended as something serious. I am very proud of the work I did there. At first I thought to make this bot for myself, but later it was useful to one client. Although the quality of the code suffers, but in the near future I will redesign its architecture.

In addition to standard functions, this bot has music playback, like in music bots. As well as a mini-game for keeping online on the server using SQL databases.

This uses the already obsolete DiscordPy library, now you can easily use it replace with Nextcord.

In addition to all of the above, it has all the features for server administration.
